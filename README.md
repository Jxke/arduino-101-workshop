# Arduino 101 Workshop

Arduino example sketches for the Arduino 101 Workshop by Jake Tan.

## Sketches

- `Blink`: basic LED blink example.
- `AnalogRead`: reads the analog value from a sensor on `A0`.
- `Combine`: combines analog sensor reading with green/red LED feedback to show when an object is in range or too close.
